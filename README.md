# sahara
Daily Check-in

# venv
```
# Create venv
python3 -m venv venv
# Activate venv
source venv/bin/activate
# Exit venv
deactivate
```

# Install
```
pip install --upgrade pip
pip install -r requirements.txt
```

# Plugin
```
cd sahara_auto/extensions/
unzip okx.crx -d okx/
```

# Run
```
cp conf.py.sample conf.py
python3 sahara.py
```
